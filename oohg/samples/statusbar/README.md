# StatusBar

* demo1.prg Ejemplo de creación de un StatusBar, asignación de texto y acción a ejercer al recibir un click
* demo2.prg Mismo que demo1.prg
* demo3.prg Ejemplo de statusbar con KEYBOARD, DATE  y CLOCK 
* demo4.prg Ejemplo de como asignar acciones a los items en tiempo de ejecución
* s001.prg This sample shows how place a ToolBar at the bottom of a form that also has a Statusbar.
* s002.prg This sample shows how to place a StatusBar control in a window that has virtual height.
* s003.prg This sample shows how to place a progressbar control "inside" a statusbar control.
* s004.prg This sample shows how to place a progressbar control "inside" a statusitem and how to replace it when the statusitem's width is changed.
* s005.prg This sample shows how to place a LABEL control "inside" a statusbar and how to resize it when the statusbar's width is changed.
* s006.prg This sample shows how to change the font.
