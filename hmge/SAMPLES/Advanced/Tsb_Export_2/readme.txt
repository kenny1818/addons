MINIGUI - Harbour Win32 GUI library Demo

ОПИСАНИЕ:
~~~~~~~~~~~~
Тестовый пример работы с объектом Tsbrowse для МиниГуи:

1) Таблица по dbf и массиву. Настройки таблицы.
2) Использование типов полей в базе '^', '+', '=', '@', 'T'
3) Использование вспомогательного класса TSBcell для быстрого экспорта данных.
4) Использование нумератора в таблице.
5) Экспорт таблицы Excel, Word, Open Office в файлы: xls/xml/doc
6) Поддержка формул в таблице для рассчёта в самом Excel - текст в формулу.

РАСПРОСТРАНЕНИЕ:
~~~~~~~~~~~~~
Эта программа является бесплатной, и может свободно копироваться и распространяться,
до тех пор, как она не изменяется каким-либо образом, и никакая плата не взимается
за распределение (плата за небольшую доставку и обработку приемлемо).

DESCRIPTION:
~~~~~~~~~~~~
Test example of working with Tsbrowse object for MiniGui:

1) Table by dbf and array. Table settings.
2) Using field types in the base '^', '+', '=', '@', 'T'
3) Using the TSBcell helper class for fast data export.
4) Using the numerator in the table.
5) Export Excel, Word, Open Office tables to files: xls / xml / doc
6) Support for formulas in a table for calculation in Excel itself - text in a formula.
