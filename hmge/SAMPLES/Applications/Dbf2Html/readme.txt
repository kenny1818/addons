   DBF to HTML Wizard
  Version 1.02 Freeware
-------------------------
Author:   Grigory Filatov

The 'DBF to HTML' program allows you to convert your dbf files
to HTML pages. You can define such parameters as border width,
cellspacing or cellpadding.
